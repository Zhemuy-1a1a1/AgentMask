## 使用方式

1. 打开对应的合集页
2. 找到你想用的 Agent
3. 点击链接打开对应的 `.md` 文件
4. 复制全文 → 粘贴到你的 AI 工具（ChatGPT / Claude / Kimi 等）
5. 发送你的需求，开始使用

---

## 为什么是「导航」不是「打包合并」？

AgentMask 的核心设计是**一个 Agent = 一个 .md 文件 = 直接导入使用**。

合集页不是把内容合并成一个大文件，而是帮你快速找到适合你的 Agent。每个 Agent 仍然独立，可单独使用、单独更新、单独付费。
