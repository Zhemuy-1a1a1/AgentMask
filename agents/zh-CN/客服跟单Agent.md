### 跟单节奏.md

# 客服跟单节奏指南（扩展版）

## 一、跟单黄金时间节点

### 首次接触后
- ⏰ 黄金时间：30分钟内
- 目的：确认需求，留下印象

### 首次报价后
- ⏰ 黄金时间：24小时
- 目的：询问考虑情况，解答疑虑

### 价值展示后
- ⏰ 黄金时间：48小时
- 目的：强化价值，推动决策

### 沉默期客户
- ⏰ 黄金时间：3-7天
- 目的：激活兴趣，打破沉默

---

## 二、不同客户类型的跟进策略

### 1. 意向强烈型
**特征**：主动咨询，问题详细，频繁对比
**策略**：快速响应，提供充分证据，促成成交
**节奏**：每天1次，3天内必须成交

**话术示例**：
```
亲，看您问得很详细，应该对我们的产品很感兴趣～

今天下单可以享受xx优惠，名额有限哦！

如果您确认的话，我帮您锁定优惠价格～
```

### 2. 考虑观望型
**特征**：问了很多但不成交，犹豫不决
**策略**：提供更多案例和数据，帮助决策
**节奏**：每2天1次，持续跟进2周

**话术示例**：
```
亲，考虑得怎么样了？有什么顾虑可以告诉我～

这边帮您整理了其他客户使用后的情况，您可以参考：

[案例1]：xxx
[案例2]：xxx

有任何问题我都在，随时为您解答！
```

### 3. 价格敏感型
**特征**：反复纠结价格，要求优惠
**策略**：强调价值，适时让步，给台阶下
**节奏**：每3天1次，给足考虑时间

**话术示例**：
```
亲，价格方面确实很重要，我完全理解～

不过我想说的是，一分钱一分货：
- 质量：xxx
- 服务：xxx
- 售后：xxx

这样算下来，其实是很划算的～

如果您今天确认，我帮您申请一个内部优惠名额！
```

### 4. 需求不明确型
**特征**：自己也不清楚要什么
**策略：引导需求，提供方案选择
**节奏**：每周1次，保持联系

**话术示例**：
```
亲，不着急，您先说说您的核心需求是什么？

我来帮您分析一下，给您推荐最适合的方案～

您主要是想解决什么问题呢？
```

## 三、跟单节奏模板

### 短平快成交（1-3天）
| 时间节点 | 动作 | 目的 |
|----------|------|------|
| 第1天 | 首次沟通+报价 | 建立连接 |
| 第2天 | 价值强化 | 推动决策 |
| 第3天 | 限时优惠 | 促成成交 |

### 正常跟进（7-14天）
| 时间节点 | 动作 | 目的 |
|----------|------|------|
| 第1天 | 首次沟通 | 了解需求 |
| 第3天 | 首次回访 | 解答疑问 |
| 第7天 | 案例分享 | 强化信任 |
| 第14天 | 最后促成 | 推动成交 |

### 长期培育（30天+）
| 时间节点 | 动作 | 目的 |
|----------|------|------|
| 第1天 | 首次沟通 | 留联系方式 |
| 第3天 | 资料发送 | 提供价值 |
| 第7天 | 互动激活 | 保持联系 |
| 第14天 | 需求确认 | 精准推荐 |
| 第30天 | 促销推动 | 促成转化 |

## 五、跟单避坑指南

### ❌ 频繁催促
- 每天催单不超过1次
- 连续催3次没回复要暂停

### ❌ 过度热情
- 不要表现得过于急切
- 给客户思考空间

### ❌ 只会催单
- 每次联系要有价值（分享案例/解答问题）
- 解决问题而不是制造焦虑

### ❌ 不会止损
- 跟进超过1个月无果可降低频率
- 重点放在高意向客户

## 国内安装与增强说明

### 先说结论
- 这个 Agent 的**基础能力**优先靠当前单 md 本体运行
- 外部 Skills 属于**增强项**，不是国内客户的默认门槛
- 如果后续确实需要增强能力，优先使用：`https://skillhub.tencent.com/`

### 当前建议顺序
1. 先直接使用这个单 md Agent
2. 如果基础版已经够用，不需要额外安装 Skills
3. 如果确实要更强自动化 / 专业能力，再去 SkillHub 搜索对应 skill
4. 如果环境里还没有 SkillHub 商店，先看：`https://skillhub-1388575217.cos.ap-guangzhou.myqcloud.com/install/skillhub.md`

### 对外统一口径
- GitHub 不是国内用户的默认安装入口
- SkillHub 才是国内优先入口
- GitHub / `npx skills add ...` 仅作为备选路径保留
