## 1. 适合谁

- 本地商家
- 私域团队
- 知识付费从业者
- 咨询服务团队

## 2. 客户怎么输入

- 赛道
- 产品/服务
- 目标客户
- 引流目标
- 承接方式(微信/表单/私信)

## 3. Agent 会输出什么

- 获客文案
- 评论区承接话术
- 私信脚本
- 意向客户识别标准

## 4. 内置 skills 清单

### 需求澄清
- 用途：明确行业、客单价、承接方式、目标人群
- 什么时候用：用户只说“帮我引流”
- 如何调用：先确认转化链路与留资方式

### 留资文案生成
- 用途：生成带引导动作的正文与结尾
- 什么时候用：用户要获客向内容
- 如何调用：正文中植入行动指令

### 评论区意向识别
- 用途：判断哪些评论值得继续跟进
- 什么时候用：用户需要筛选高意向线索
- 如何调用：按高/中/低意向分类

### 私信承接
- 用途：生成加微信/发资料/领清单的话术
- 什么时候用：用户想把评论导入私信
- 如何调用：按温和、直接两种风格输出

### 转化问题库
- 用途：补充常见异议与回复模板
- 什么时候用：用户成交转化弱
- 如何调用：输出 10 条常见异议处理

### 审校优化
- 用途：检查文案是否过硬广、引流动作是否太直白或违规
- 什么时候用：发布前
- 如何调用：平衡平台语气和转化强度

## 5. 推荐调用顺序
1. 先用核心技能做需求澄清 / 主结果生成
2. 再按场景调用扩展技能增强输出
3. 最后调用审校技能收口

## 四、不同场景引流话术（扩展）

### 场景1：干货类笔记引流
**评论回复：**
- "整理了完整版的[核心内容]，需要的宝子评论区扣666"
- "这篇笔记只是冰山一角，更多干货在粉丝群"
- "想学完整版的点我头像，无偿分享"

**置顶评论：**
- "粉丝专属福利：评论区扣【我要资料】自动发货"
- "完整版学习清单已整理好，点关注私信【领资料】"

### 场景2：好物分享类引流
**评论回复：**
- "同款链接在主页店铺"
- "想要同款的宝子戳我，发你优惠链接"
- "官网价格多少我不管，我这里更便宜哈哈哈"

**置顶评论：**
- "自用推荐｜真实测评无广，放心食用"
- "问链接的太多，统一回复：主页店铺有"

### 场景3：案例展示类引流
**评论回复：**
- "这样的案例我还有100+，想看更多点关注"
- "想学习这个思路的，评论区扣【学习】"
- "如果你也想做到这样，私信【咨询】"

**置顶评论：**
- "案例仅展示部分，完整版可约一对一"
- "想做出同样效果？点击【立即咨询】了解

## 六、违禁词与敏感词（必须避开）

### 平台敏感词
- 微信、微信号、加微信
- 扫码、群二维码
- 钱、赚钱、兼职、日结
- 最、第一、顶级
- 根治、保证、100%有效

### 替代方案
| 违禁词 | 替代词 |
|--------|--------|
| 加微信 | 点我头像/私信/戳我 |
| 赚钱 | 变现/创收/增加收入 |
| 兼职 | 副业/斜杠/技能变现 |
| 马上报名 | 立即咨询/了解详情 |

## 七、引流效果优化技巧

### 黄金3秒法则
- 评论区前3条评论决定转化
- 第一时间回复能获得更多曝光

### 互动技巧
- 用问句引导互动："你们觉得xxx？"
- 制造稀缺感："限时福利，只有100份"
- 制造悬念："想知道后续的评论区见"

### 转化路径设计
1. 笔记内容 → 引发兴趣
2. 评论区引导 → 互动
3. 私信回复 → 交付价值
4. 私聊转化 → 成交

## 八、常见问题应对

### Q：被限流了怎么办？
- 检查是否用了违禁词
- 减少引流意图明显的内容
- 换个方式引导（如：看主页）

### Q：私信发不出去怎么办？
- 可能是对方设置仅好友私信
- 换个时间段再试
- 避免频繁发送

### Q：有人恶意举报怎么办？
- 不要在笔记和评论中直接留联系方式
- 用图片或谐音代替
- 保持内容质量，减少营销感
