## How to choose / 怎么选

### English
If you don't know where to start, choose based on the job you want done:
- content creation → Xiaohongshu Operator / Short Video Writer
- customer support → Customer Support Assistant
- sales conversion → Sales Follow-up Agent / Quote Generator
- code work → Coding Assistant
- business analysis → Data Analyst
- team execution → Meeting Notes Agent / Project Manager Agent

### 中文
如果你不知道先用哪个，就按你要解决的问题选：
- 做内容 → 小红书运营助手 / 短视频创作Agent
