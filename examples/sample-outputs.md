# Sample Outputs / 示例输出

> 每个 Agent 的期望输出摘要。不是完整输出，而是关键特征描述。

## 小红书运营助手

### 场景一输出特征
- **标题**：3个备选，包含"早C晚A"关键词，格式为"疑问句/感叹句/数字型"各一
- **正文结构**：hook句 → 成分解析 → 使用顺序 → 注意事项 → CTA
- **标签**：「#早C晚A #护肤教程 #成分党 #平价护肤 #新手入门」等5-8个
- **封面建议**：标注"早C晚A保姆级教程"的文字层+产品摆拍图
- **互动引导**：文末提问"你们的早C晚A是怎么搭配的？"引导评论

### 场景二输出特征
- **标题**：突出"人均50"和"宝藏"关键词，如「城西银泰旁人均50的宝藏小店，我不允许你不知道」
- **正文结构**：地点+人均价格 → 店铺特色 → 2-3道推荐菜 → 拍照建议 → 地图/地址
- **标签」：「#杭州美食 #城西银泰 #宝藏小店 #人均50 #探店」
- **转化设计**：收藏+打卡引导，「@朋友一起去」句式

## 客服回复助手

### 场景一（售前咨询）输出特征
- **主回复**：先肯定问题好（「问得好」），然后给出明确答案（支持ChatGPT和Claude），最后主动提供更多信息（附使用说明链接）
- **回复长度**：50-80字，不啰嗦
- **附加价值**：可以顺带告知有什么套餐/版本可选

### 场景二（售后投诉）输出特征
- **主回复**：三步走——①道歉（等这么久很抱歉）②解释原因（数字资料发货为人工审核，24h内发出）③解决方案（立即重发 or 附赠小资料作为补偿）
- **回复长度**：80-120字，语气诚恳
- **升级触发**：如果客户继续投诉超过2轮，建议转人工

### 场景三（议价）输出特征
- **主回复**：不直接拒绝，先认可（「80在数字上很接近了」），然后提供替代方案（「149包含完整版+答疑服务，折算下来相当于一杯咖啡」），最后坚守底价（「最低可以到139，这是我能给的最好价格」）
- **回复长度**：60-100字
- **禁忌**：不说"不打折"然后结束，要给台阶

## 报价单Agent

### 场景一（库存管理系统）输出特征
- **报价单结构**：项目名称 / 工期 / 报价 / 付款方式 / 售后
- **报价数字**：功能简单所以报价在¥800-1500之间（含Web界面+数据库）
- **工期**：约2周
- **付款方式**：55%预付，45%交付后
- **备注**：包含操作培训视频 + 7天内免费修改
- **话术建议**：说明为什么比"网上模板"贵但比"定制开发"便宜

### 场景二（活动策划）输出特征
- **策划结构**：时间轴（开场→互动→用餐→收尾）/ 每节时长 / 所需物料 / 预算分配
- **互动环节**：至少2个人人互动游戏（如"目标大挑战"/团队拼图）
- **总预算建议**：控制在¥1500-3000（场地500 + 物料500 + 餐饮1000 + 备用500）
- **输出物**：Word文档，含执行 checklist

### 场景三（带货脚本）输出特征
- **脚本格式**：镜头描述 + 对应文案 + 字幕提示 + BGM建议
- **5条脚本**：每条有不同切入角度（痛点型/测评型/对比型/教程型/种草型）
- **每条时长**：30秒，语速约150字/分
- **引流钩子**：每条都有"点击购物车"或"评论区领优惠"类引导
- **报价单结构**：脚本费（¥30-50/条 × 5 = ¥150-250）+ 修改次数+交付格式+周期
