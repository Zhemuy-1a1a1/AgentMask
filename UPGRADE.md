# Upgrade Path / 升级路径

## English
- Open-source version: lightweight public starter, still based on one markdown file
- Standard version: a stronger single-agent version, still delivered as one core `.md` file
- Advanced (custom) version: deeper adaptation for industry, workflow, memory strategy, or long-term use, while keeping one `.md` file as the core product

## 中文
- 开源版：轻量公开入口，核心仍然是一个单 md 文件
- 标准版：单个 Agent 的更完整正式版，核心仍然是一个单 md 文件
- 进阶（定制）版：按行业、业务流程、记忆策略、长期使用场景做深度适配，但核心仍然保持单 md 文件
