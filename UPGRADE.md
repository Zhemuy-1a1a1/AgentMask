# Upgrade Path / 升级路径

## English
- **Public standard edition**: complete enough to copy, import, and use directly; focused on clear structure and broad applicability
- **Optimized version**: stronger production usability, lower API cost, model compatibility tuning, better structured outputs, and more China-friendly setup
- **Advanced (custom) version**: deeper adaptation for industry, workflow, memory strategy, automation, or long-term use, while keeping one `.md` file as the core product

## 中文
- **公开标准版**：已经足够用于直接复制、导入和开始使用，重点是结构清晰、适用面广
- **增强优化版**：更强调真实生产可用性、更低 API 成本、模型兼容调优、输出结构稳定，以及更友好的国内源适配
- **进阶（定制）版**：按行业、业务流程、记忆策略、自动化链路、长期使用场景做深度适配，但核心仍然保持一个 `.md` 文件
